Important details:

This plug-in needs jQuery to run. All parallax objects are initialized automatically, just add a 'background-image' and the class 'parallaxObject' and done!
You can have as many parallaxObjects as you want in the same page.
Because we use 'background-attachment:fixed' for smooth results, the image adapts to the size of the screen, so all images should be large for the plug-in to work.
How much the background moves will depend on the size of the image, unless added the attribute 'data' with a value such as data="0.1".
The parallaxObjects will stop executing when the parallaxObject is no longer in the screen to prevent overheading
You literally don't have anything else to do and it'll work like charm in any scenario.





















Demo two:

Simulating depth by animating selected children items within the parallaxObject at different speeds using 'data' attribute

Important details:

Create items within the parallaxObject that respond to the effect by adding the class 'parallaxChild' into every item you want to animate.
The optional attribute 'depth' can be added so that the plug in knows which children elements are "closer" and which are "farther" so they can move at different speeds, just like it would happen in real life.
To correct positioning of the children simply add some 'margin-top' to them.
Set max-width: 100%; to prevent moving texts from expanding out of the screen.




















Demo three:

Activating the perspective setting to enhance depth perception


Static title of the banner

Important details:

You can have moving and non-moving items within a parallaxObject. As mentioned above simply add class="parallaxChild" to the ones you want to be animated.
A perspective option has been created so that children will be resized (width and height proportionally) based on their specified depth. Simply add the class 'perspective' such as class="parallaxChild perspective". You will need to specify 'depth' to make this option work properly.
You can play around with the values and even use decimals such as 1.5 in the 'depth' attribute to control how the move and scale
If the 'perspective' has been added, the property "background-size:cover" will be automatically added to make the perspective effect work.
